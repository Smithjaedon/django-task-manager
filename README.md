# Django Task Manager

A full-stack task management web application built with Django. This project was built to develop a strong understanding of Django's core concepts, including authentication, ORM queries, class-based views, and pagination.

## Features

- **User Authentication** – Register, log in, and log out securely. Users can only view and manage their own tasks.
- **CRUD Operations** – Create, read, update, and delete tasks.
- **Task Statuses** – Assign statuses to tasks: To Do, In Progress, or Done.
- **Search & Filtering** – Search tasks by name and filter by status.
- **Sorting** – Sort tasks by title, creation date, or last updated date in ascending or descending order.
- **Pagination** – Tasks are paginated for a clean, organized browsing experience.

## Tech Stack

- **Backend:** Python, Django
- **Frontend:** HTML, CSS, Tailwind CSS
- **Database:** SQLite
- **Templating:** Django Templates
